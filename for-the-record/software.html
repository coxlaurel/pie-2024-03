Software Systems Description

There are two main software components for this project. One of them is C# code loaded onto an Arduino (with the Adafruit Motor Shield) that interacts with firmware to control and track different aspects of our record player. The other is CircuitPython code that works with serial communication signals sent from the Arduino to the Matrix Portal M4 and translates them into text that gets scrolled across the Adafruit RGB Display.
