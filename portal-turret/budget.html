Budget

Listing the resources and money spent to bring this project to life.

We were given $250 to spend on materials and components to build the Portal Turret.

Bill of Materials

Name Description Price
Raspberry Pi Camera 4 Model B A camera for the Raspberry Pi 4. Used for motion detection and pose estimation. 63.99
Raspberry Pi Camera Module v2 The single board computer used to execute code and control various sensors and motors on the Portal Turret. 12.49
128GB SD Card A flash memory card used for the Raspberry Pi to store the Raspian OS and other files. 13.80
Adhesive Rubber Sheets Adhesive rubber sheets to attach the bottom of the feet to prevent slipping.
Foam Foam to create the outer shell.
Zip Ties Used to keep components in place.
Screws and Bolts* Screws for screwing components together or to fix them in place.
Fiberboard* Used to create the internal layers within the chassis.
Wood* Used to create the chassis frame and legs.
JBL GO 2* A speaker to play the audio from the Raspberry Pi 4. 36.79
LSM303AGR* A 3-axis accelerometer and magnetometer to measure the acceleration. 12.50
Sunfounder Sound Sensor Module* A sound sensor to measure intensity of sound input. 6.99
Sunfounder AD/DA Converter PCF8591 Module* A module to convert signals from digital to analog or from analog to digital. 19.99
Sunfounder Infrared-Receiver Module* A module to receive infrared signals. 6.99
Infrared Remote Control* An IR remote control to send signals to the IR receiver module. 7.87
9V DC Motor* A motor to move the wings in and out using a rack-and-pinion system. 5.99
TowerPro SG90 Servo* Two micro servo motors to rotate the wings forwards and backwards. 6.99
Raspberry Pi GPIO Breakout Board Kit* A Raspberry Pi GPIO breakout board kit, which includes a breadboard, GPIO breakout board, ribbon cable, and jumper cables. 9.99
PLA for 3D Printing* PLA filament for 3D printing various components. 13.99
